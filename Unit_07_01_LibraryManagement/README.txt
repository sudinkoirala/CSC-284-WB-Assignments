##  Overview
This project implements a simple **Library Management System** using object-oriented programming in C++.  
It manages:
- Books (title, author, year, ISBN, availability)
- Members (name, ID, borrowed books)
- Library operations (add, borrow, return, display)

---

##  Files
| File | Description |
|------|--------------|
| `Book.h` | Defines the Book class |
| `Member.h` | Defines the Member class |
| `Library.h` | Declares Library functions |
| `Library.cpp` | Implements Library functions |
| `main.cpp` | Contains menu-driven user interface |

---

##  Features
✅ Add and remove books  
✅ Register members  
✅ Borrow and return books  
✅ View all books and members  
✅ Uses `std::vector`, iterators, and algorithms  

---

##  How to Build and Run in Xcode
1. Open **Xcode → File → New → Project → macOS → Command Line Tool**
2. Set **Language** to **C++**
3. Add all `.h` and `.cpp` files to the project
4. In **Build Settings → C++ Language Dialect**, set to **C++20**
5. Click ▶️ **Run** to execute
6. Follow the on-screen menu
